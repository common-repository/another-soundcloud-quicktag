== Description ==
Another SoundCloud Quicktag. Establish a new button in the tinymce editor, so you can easy link to SoundCloud files. The plugin works with multiple links in one post... You have various options

Tested with: Safari 5.1, Firefox 6 and Chrome 13 under Mac OS X 

= Thanks = 
Many thanks to Johannes Wagener for [SoundCloud Shortcode (ver 1.1.9)](http://wordpress.org/extend/plugins/soundcloud-shortcode/)

== Installation ==
To install:
1. Upload the 'another-soundcloud-quicktag' folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use it in the tinymce. 


== Upgrade Notice ==
Just upgrade. Just overwrite the folder. No problems expected :)
